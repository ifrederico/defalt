# Contributing

Thanks for your interest in contributing to Defalt!

## Ways to Contribute

- **Bug reports**: Open an issue with steps to reproduce
- **Feature requests**: Open an issue describing the use case
- **Pull requests**: Fork, make changes, submit PR

## Development Setup

```bash
cd dashboard
cp .env.example .env  # Configure your Ghost credentials
bun install
bun run dev
```

## Pull Request Guidelines

1. Fork the repo and create your branch from `main`
2. Run `bun run lint` and `bun test` before submitting
3. Keep PRs focused - one feature or fix per PR
4. Update docs if you change behavior

## Questions?

Open an issue - we're happy to help.
